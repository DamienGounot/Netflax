# netflax

## npm & vue install
https://nodejs.org/en/download/
npm install -g @vue/cli

## Project creation
vue create netflax

## Vuetify install inside project
cd netflax
vue add vuetify
vue --version
vue ui

## axios & router install 
npm install axios
npm install --save vue-axios
npm install --save vue-router

## Running for dev
npm run serve

## Build
npm run build
